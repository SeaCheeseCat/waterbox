# 失败的制作者

## 样式

本篇目下所有的bug都按照这个格式书写:

???+ bug "这是一个bug"

    很遗憾的是它没有被修复！

如果一个bug被修复了，那么它就会被移到已解决列表中，并且变为已解决的样式：

???+ success "这是一个已解决bug"

    它被修复了！
